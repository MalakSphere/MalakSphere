Project: My Favorite Recipe

  •   Description:Create a single HTML page that shows your favorite recipe with title, image, ingredients, and steps.
  •  Steps:
    1. Create index.html
    2. Add HTML boilerplate (<!DOCTYPE html>, <html>, <head>,<body>).
    3. . Inside <head>, put a <title> like “My Favorite Recipe”.
    4. In <body>:
      • Add <h1> for the recipe name.
      • Add an <img > showing the dish
      • Write a short <p> description.
      • Create a <h2> Ingredients section with a <ul> list.
      • Create a <h2> Steps section with an <ol> list.
      • Add a link <a> to another recipe website for reference
    5. Add a form for user input.
      • Create a <h2> heading: “Share Your Recipe”
      • Inside a <form>, add:
        • <label> + <input type="text"> for recipe name.
        • <label> + <textarea> for description.
        • <label> + <input type="url"> for image link.
        • <button type="submit"> to send.
    6. Save and open the browser.
